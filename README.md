# Bound_Ball
マウスを使ってボールを弾くゲーム

## 遊び方
### 目的
ゴール(渦巻き)にボールを運ぶ
### 操作フロー
マウス左ボタン長押し->マウスを弾きたい方向に移動->左ボタンを離す->ボールがその方向に弾かれる
